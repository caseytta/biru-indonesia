Konverter CSV → 3-galeridata.js

Cara pakai:

  1. Di Google Sheets, buka tab proyek (Tabel 2). Pastikan baris pertama header dengan kolom (urutan bebas, nama harus cocok):
    NamaProjek, Tahun, SolusiAplikasi, ProductSegment, ProductSegment_Brand, JenisMaterial, Img
  2. File → Download → CSV.
  3. Pilih file CSV di bawah ini.
  4. Salin hasil di kotak teks, lalu tempel menggantikan array PROJECTS di js/3-galeridata.js.

Kolom Img cukup nama file (mis. arumaya.jpg). File gambarnya taruh di asset/3-galeri/.
